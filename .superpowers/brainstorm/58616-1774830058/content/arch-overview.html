系统架构 · 方案 B（并发 tick + Memory Stream）

三层结构：模拟引擎 → FastAPI 服务 → 浏览器前端

⏱ 一个 Tick 的流程
1. Scheduler 触发（每 4 秒）
2. 读取 World State → 构建 4 个 prompt
3. asyncio.gather 并发调用 Claude × 4
4. 解析返回：目标房间 + 对话文本 + 内心独白
5. Pathfinder 计算移动路径帧
6. 更新 World State → WebSocket 广播
7. 前端 Canvas 播放移动动画 + 弹出气泡
📦 LLM 返回格式（JSON）
{
  "思考": "无限在厨房忙了很久，
    该去叫大家吃饭了",
  "目标房间": "餐厅",
  "动作": "走向餐厅，招呼众人",
  "对话": "饭好了！大家快来！",
  "情绪": "开心"
}
